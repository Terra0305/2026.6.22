# AquaGuard AI 해커톤 데모

## 실행

```bash
node local-server.mjs
```

- 왼쪽 브라우저: `http://localhost:4173/simulator`
- 오른쪽 브라우저: `http://localhost:4173/`

두 화면은 반드시 같은 호스트와 포트로 열어야 실시간 상태가 공유됩니다.

## 시연 순서

1. 두 화면을 나란히 연다.
2. `/simulator`에서 `브라우저 알림 허용`을 누른다.
3. 펌프 부하 증가 또는 베어링 마모 발생을 누른다.
4. 정상 → 부하 증가 → 위험까지 약 6.5초 동안 수압·진동·그래프가 변하고 임계값에 따라 상태가 전환되는 모습을 확인한다.
5. 복구 시작을 누르면 센서값이 점진적으로 정상 범위로 내려간다.
6. 긴급 위험에서는 오른쪽 모바일 프레임 안에 전체 화면 긴급 경고가 표시된다.

브라우저 알림은 `localhost`에서 실행하고 권한을 허용해야 표시됩니다.

# AquaGuard AI 해커톤 데모

## 실행

```bash
node server.mjs
```

- 왼쪽 브라우저: `http://localhost:4173/simulator`
- 오른쪽 브라우저: `http://localhost:4173/`

두 화면은 반드시 같은 호스트와 포트로 열어야 실시간 상태가 공유됩니다.

## 시연 순서

1. 두 화면을 나란히 연다.
2. `/simulator`에서 `브라우저 알림 허용`을 누른다.
3. 정상 → 주의 → 긴급 위험 → 복구 시나리오를 차례로 누른다.
4. 기존 홈·장비 상세·차트의 수치와 상태 변화를 확인한다.
5. 긴급 위험에서는 오른쪽 모바일 프레임 안에 전체 화면 긴급 경고가 표시된다.

브라우저 알림은 `localhost`에서 실행하고 권한을 허용해야 표시됩니다.
